# Hire-to-Retire-H2R--Project

## 📌 Project Overview

This project demonstrates the complete implementation of the Hire-to-Retire (H2R) process using SAP HR (Human Capital Management). It covers the entire employee lifecycle from recruitment and hiring to payroll management, performance evaluation, and final exit or retirement.

The system provides a centralised and integrated approach to managing employee data and HR processes efficiently.

---

## ❓ Problem Statement

In many organisations, employee management is handled through multiple disconnected systems or manual processes. This leads to:

- Data inconsistencies across departments
- Payroll errors and delays
- Lack of transparency in performance tracking
- Difficulty in maintaining employee records
- Inefficient HR operations

This project addresses these challenges by implementing a unified SAP-based solution that ensures consistency, automation, and real-time data management.

---

## ⚙️ Implementation Steps & SAP T-Codes

The Hire-to-Retire process is executed through the following key stages:
| # | Step | T-Code | Description |
| --- | --- | --- | --- |
| 1 | **Recruitment** | `PB10 / PB20` | Manage applicants and recruitment process |
| 2 | **Hiring** | `PA40` | Create employee records in SAP |
| 3 | **Onboarding** | `PA30` | Maintain employee master data |
| 4 | **Payroll** | `PC00_M40_CALC` | Process salary and payroll calculations |
| 5 | **Performance** | `PHAP_ADMIN` | Manage performance appraisal |
| 6 | **Exit** | `PA40` | Handle employee separation and exit |
